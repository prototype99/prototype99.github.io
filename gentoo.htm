how to install gentoo

  1. boot into install media (calculate linux has a larger number of options and better config options)
  2. su && passwd
  3. connect to internet
  4. set up disk with a gpt table. add swap, then boot partition, then data partition (generally with sata xfs). you may also label and name these partitions, it does make things easier. if you aren't using efi you also need a 1 mib bios_grub partition
  5. sudo swapon /dev/sd[*]1 && sudo mkdir /mnt/gentoo && sudo mount /dev/sd[*]3 /mnt/gentoo && cd /mnt/gentoo where [*] is the drive letter
  6. sudo git clone https://github.com/prototype99/prototype99.git
  7. /var/db/repos/prototype99/install.sh
  8. that's it really, the script is designed to guide you through the entirety of the remaining steps
