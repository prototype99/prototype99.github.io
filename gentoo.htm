how to install gentoo

  1. boot into install media (calculate linux has a larger number of options and better config options)
  2. su && passwd
  3. connect to internet
  4. set up disk with a gpt table. add swap, then boot partition, then data partition (generally with sata xfs). you may also label and name these partitions, it does make things easier. if you aren't using efi you also need a 1 mib bios_grub partition
  5. sudo swapon /dev/sd[*]1 && sudo mkdir /mnt/gentoo && sudo mount /dev/sd[*]2 /mnt/gentoo && cd /mnt/gentoo where [*] is the drive letter
  6. go to https://www.gentoo.org/downloads/#other-arches and wget the correct download link
  7. tar xpvf stage3-*.tar.bz2 --xattrs-include='*.*' --numeric-owner
  8. rm -v -f stage3-amd64-* or reserve it as a backup
  9. add in the pre-install make.conf

10. cd ~ && mkdir --parents /mnt/gentoo/etc/portage/repos.conf && cp /mnt/gentoo/usr/share/portage/config/repos.conf /mnt/gentoo/etc/portage/repos.conf/gentoo.conf && cp --dereference /etc/resolv.conf /mnt/gentoo/etc/ && sudo mount --types proc /proc /mnt/gentoo/proc && sudo mount --rbind /sys /mnt/gentoo/sys && sudo mount --make-rslave /mnt/gentoo/sys && sudo mount --rbind /dev /mnt/gentoo/dev && sudo mount --rbind /dev /mnt/gentoo/dev && sudo chroot /mnt/gentoo /bin/bash && source /etc/profile && export PS1="(chroot) ${PS1}" && sudo mount /dev/sda3 /boot && emerge-webrsync && emerge --sync && eselect news read

11. eselect profile list

12. eselect profile set the appropriate profile

13. emerge dev-perl/XML-Parser sys-kernel/genkernel-next --ask --verbose --update --deep --newuse @world && ln -sf /proc/self/mounts /etc/mtab && genkernel kernel
