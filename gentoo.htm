how to install gentoo

  1. boot into install media (calculate linux has a larger number of options and better config options)
  2. su && passwd
  3. connect to internet
  4. set up disk with a gpt table. add swap, then data partition (generally with sata xfs) then boot partition
  5. swapon /dev/sda1 && mount /dev/sda4 /mnt/gentoo && cd /mnt/gentoo
  6. go to https://www.gentoo.org/downloads/#other-arches and wget the correct download link
  7. tar xpvf stage3-*.tar.bz2 --xattrs-include='*.*' --numeric-owner
  8. rm -v -f stage3-amd64-* or reserve it as a backup
  9. add in the pre-install make.conf
  10. cd ~ && mkdir --parents /mnt/gentoo/etc/portage/repos.conf && cp /mnt/gentoo/usr/share/portage/config/repos.conf /mnt/gentoo/etc/portage/repos.conf/gentoo.conf && cp --dereference /etc/resolv.conf /mnt/gentoo/etc/ && sudo mount --types proc /proc /run/media/guest/CLDL-18.12/proc && sudo mount --rbind /sys /run/media/guest/CLDL-18.12/sys && sudo mount --make-rslave /run/media/guest/CLDL-18.12/sys && sudo mount --rbind /dev /run/media/guest/CLDL-18.12/dev && sudo mount --rbind /dev /run/media/guest/CLDL-18.12/dev && sudo chroot /run/media/guest/CLDL-18.12 /bin/bash && source /etc/profile && export PS1="(chroot) ${PS1}" && sudo mount /dev/sda3 /boot
